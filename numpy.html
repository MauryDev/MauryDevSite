NumPy básico para iniciantes

 

Introdução

 

A biblioteca NumPy é uma ferramenta muito poderosa para ciência de dados que é uma abreviatura de Numerical Python e por isso é usada bastante por usuários de ciência de dados, por causa do suporte e velocidade para grande demanda de dados que é chamada de BigData.

 

Nessa pequena introdução veremos métodos da NumPy e seu uso para iniciantes de cientistas de dados mesmo que alguns preferiram a biblioteca pandas que é uma evolução do NumPy, mas ainda assim ela não perdeu seu poder na área de DS.

 

Ndarray

 

Ndarray é um classe do NumPy para array n-dimensional de valores do mesmo tipo e essa é uma ferramenta poderosa para BigData e é essencial saber explorar.

Na API da NumPy existem vários métodos para criação, tratamento,cópia e de obter uma informação de um Ndarray que é usado para armazenar um grande escala de dados nos tópicos seguintes.

Para criar um ndarray da forma mais simples necessita  atribuir o valor do parâmetro shape que é a tuplas de números inteiros para as dimensões do array.

Exemplo:

import numpy as np

dadostest = np.ndarray(shape=(2,2))

print(dadostest)

Output:

array([[5.e-324, 5.e-324],

       [5.e-324, 0.e+000]])

 

numpy.random.rand

Esse método retorna um ndarray com valores aleatório.

Exemplo:

dados = np.random.rand(2,3)
print(dados)

 

Output:

array([[0.23993936, 0.81819481, 0.19388717],

       [0.41966829, 0.24681328, 0.12850882]])

 

numpy.ndarray.dtype

Nesse atributo retorna o tipo.

Exemplo:

print(dados.dtype)

 

Output:

dtype('int64')

 

numpy.dtype

 

Os tipos de valores suportando para construção de array no numpy são:

·         Números sem ponto que pode contém sinal ou não.

o    Byte

o    Unsigned Byte

o    Integer

o    Unsigned Integer

·         Números flutuantes de 32 bits e 64 bits que são chamada float (float32) e double (float64) no C++.

o    Float

o    Double

·         Valores lógicos que representa se é verdadeiro (1) ou falso (0) que é chamado de boolean em várias linguagens de alto nível.

·         Unicode String que são 16 bits para representa um caractere.

·         DateTime é usado para data e horas.

o    np.datetime64

·         Objetos do Python

·         Raw Data (void)

·         Zero-terminated bytes (é um tipo não recomendável usar)

·         TimeDelta é usado para representar dia ,hora,semana ou outra forma para representa o tempo.

o    np.timedelta64

Numpy.arange

 

Essa função criar um ndarray que tem valores gerado a partir de valores dentro em  um intervalo sendo o intervalo final não sendo incluindo .

numpy.ndarray.astype

Essa função criar um ndarray de outra ndarray que será convertida para um determinando tipo.

Exemplo:

value = np.array([2.0,5.9,6.0])

value2 = value.astype(np.int64)

print(value)

Output:

operadores  lógicos e aritméticos do ndarray

No numpy existem operadores aritmético e lógicos para ndarray que são:

·         +

o    Somar cada valor a partir de outro valor do mesmo tipo ou outro ndarray.

·         -

o    Subtrair cada valor a partir de outro valor do mesmo tipo ou outro ndarray.

o    Inverter os sinais.

·         *

o    Multiplicar cada valor a partir de outro valor do mesmo tipo ou outro ndarray.

·         /

o    Dividir cada valor a partir de outro valor do mesmo tipo ou outro ndarray.

·         >

o    Compara cada valor se é maior a partir de outro valor do mesmo tipo ou ndarray.

·         <

o    Compara cada valor se é menor a partir de outro valor do mesmo tipo ou ndarray.

·         ==

o    Compara cada valor se é igual a partir de outro valor do mesmo tipo ou ndarray.

·          !=

o    Compara cada valor se é diferente a partir de outro valor do mesmo tipo ou ndarray.

·         >=

o    Compara cada valor se é igual ou menor a partir de outro valor do mesmo tipo ou ndarray.

·         <=

o    Compara cada valor se é igual ou menor a partir de outro valor do mesmo tipo ou ndarray.

 

Exemplo 1:

value = np.array([2,5,6])

value2 = np.array([5,2,36])

print(value + value2)

​Output:

array([ 7, 7, 42])

Exemplo 2:

value = np.array([-2,5,6])

value2 = 2

print(value + value2)

​Output:

array([0, 7, 8])

 

Exemplo 3:

value = np.array([2,5,6])

value2 = np.array([5,2,36])

print(value - value2)

​​Output:

array([-3, 3, -30])

Exemplo 4:

value = np.array([-2,5,6])

value2 = 2

print(value * value2)

print(value / value2)

​Output:

array([-4, 10, 12])

array([-1., 2.5, 3. ])

Exemplo 5:

value = np.array([2,7,5])

value2 = np.array([2,3,0])

print(value == value2)

print(value != value2)

print(value < value2)

print(value > value2)

Output:

array([True, False, False])

array([False, True, True])

array([False, False, False])

array([False, True, True])

numpy.ndarray[]

Nesse operador permitir retorna um valor ou valores de um ndarray pelo:

·         Enderenço

·         Expressão lógica

·         Intervalo

 

Exemplo:

value = np.array([2,7,5])

# Enderenço

print(value[0])

# Expressão lógica

print(value[value < 6])

# Intervalo

print(value[1:3])

​​Output:

2

array([2, 5])

array([7, 5])

 

numpy. ndarray.copy

Essa função criar um ndarray que é uma cópia de outro ndarray.

Exemplo:

value = np.array([2,7,5])

value2 = value.copy()

value2[0] = 3

print(value)

print(value2)

​Output:

array([2, 7, 5])

array([3, 7, 5])

Essa função retornar um ndarray contendo os mesmo dados com um novo modelo.

O uso de NumPy para Data Science

Nesse mini projeto vou mostrar uso do NumPy para verificar se as pessoas são maiores de idade.

1.       Criar um notebook no Google Colab.

2.       Importar o NumPy e o io.

3.       Criar uma classe tendo dois membros :

a.        Nome

b.       Idade

4.       Criar os dados a partir de um CSV que será convertido para um ndarray.

5.       Criar um algoritmo para mostrar pessoas maiores de idade.

6.       Criar um algoritmo para mostrar a quantidade de pessoas maiores de idade.

google colab

Essa ferramenta é muito importante aprender usar para um cientista de dados, porque nessa plataforma permitir criação de código em python em forma de notebook que facilitar bastante na hora de entender a lógica do algoritmo.

 

Importando o Numpy e o io

Na primeira etapa precisamos importar o NumPy para tratar os dados e o io para arquivos.

import numpy as np

import io

​Criando uma classe para o dado

Precisamos criar uma classe para representar o dado que contém as informações da pessoa que são:

·         Nome

·         Idade

 

class Usuario:

    def __init__(self,info):

        self.nome = info[0]

        self.idade = info[1]

    def __str__(self):

        return "(Nome: " +self.nome + " , Idade:" + str(self.idade)+")"

    def __repr__(self):

        return self.__str__()

criando um algoritmo para mostrar quais usuários são maiores de idade

A primeira coisa definir qual expressão de comparação para mostrar se um usuário é tem 18 anos ou mais que seria “usuario.idade >= 18” ou “usuario.idade > 17”.

def MaioresdeIdade(arr):

    return np.fromiter((

        x for x in arr if Usuario(x).idade >= 18),

        dtype=arr.dtype)

 

print(MaioresdeIdade(users))

​Output:

array([('Paulo', 18), ('Andre', 19), ('Maria', 21), ('Pedro', 24), ('Iago', 18), ('Marcos', 20), ('Leticia', 21)])

criando um algoritmo para mostrar quantos usuários são maiores de idade

Agora usaremos o membro size para obter o tamanho do ndarray depois de verificar quais pessoas são maiores de idade.

def QuantidadeMaioresdeIdade(arr):

    return np.fromiter((

        x for x in arr if Usuario(x).idade >= 18),

        dtype=arr.dtype).size

​

print(QuantidadeMaioresdeIdade(users))
